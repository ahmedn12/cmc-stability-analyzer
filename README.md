# CMC Stability Analyzer

**Pharmaceutical data science | ICH Q1E shelf-life prediction**

## Overview
Python-based stability analysis pipeline for drug product CMC development. 
Simulates ICH-compliant accelerated stability data and predicts shelf life 
using first-order degradation kinetics, Arrhenius extrapolation, and ICH Q1E 
95% confidence limit methodology.

## Methods
- First-order degradation modeling (scipy curve_fit)
- ICH Q1E shelf-life estimation (95% lower confidence limit)
- Arrhenius extrapolation across temperature conditions
- Interactive Plotly dashboard

## Tools
Python · pandas · numpy · scipy · matplotlib · plotly
